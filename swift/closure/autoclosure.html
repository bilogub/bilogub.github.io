Autoclosure argument attribute

Dec 7, 2015

While looking into some framework code saw @autoclosure argument attribute. Which I found very interesting for myself. Read about where you could apply it here.

In short – you tell compiler to wrap argument into closure implicitly and can pass this closure as an argument without curly braces.

As you can see this closure can’t take any arguments.
